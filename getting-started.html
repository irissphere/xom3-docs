Getting Started

Set up your XOM3 account and create your first AI-powered automation in under 5 minutes.

Prerequisites

Before you begin: Make sure you have Node.js 18+ installed and a XOM3 account. Sign up at app.xom3.org if you haven't already.

Quick Start Guide

Install the XOM3 SDK

Install the official XOM3 SDK using npm or yarn:

Bash 
npm install @xom3/sdk
# or
yarn add @xom3/sdk

Get Your API Key

Navigate to your XOM3 Dashboard and generate an API key.

Keep it secret! Never commit your API key to version control. Use environment variables instead.

Initialize the Client

Create a new file and initialize the XOM3 client:

JavaScript 
import { XOM3 } from '@xom3/sdk';

// Initialize with your API key
const xom3 = new XOM3({
  apiKey: process.env.XOM3_API_KEY,
  // Optional: specify region
  region: 'us-east-1'
});

Create Your First Workflow

Build an AI-powered automation workflow:

JavaScript 
// Create a customer support automation
const workflow = await xom3.workflows.create({
  name: 'Customer Support Agent',
  description: 'AI-powered customer support',
  
  // Define triggers
  triggers: [{
    type: 'webhook',
    config: { path: '/support' }
  }],
  
  // Define actions
  actions: [
    {
      type: 'ai_classify',
      config: {
        categories: ['billing', 'technical', 'general']
      }
    },
    {
      type: 'ai_respond',
      config: {
        model: 'gpt-4',
        systemPrompt: 'You are a helpful support agent...'
      }
    },
    {
      type: 'send_email',
      config: {
        to: '{{customer.email}}'
      }
    }
  ]
});

console.log('Workflow created!', workflow.id);

Test Your Workflow

Trigger your workflow with a test request:

Bash 
curl -X POST https://api.xom3.org/workflows/YOUR_WORKFLOW_ID/trigger \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "I need help with my billing", "customer": {"email": "test@example.com"}}'
Congratulations! You've created your first XOM3 automation. Check your dashboard for execution logs.
