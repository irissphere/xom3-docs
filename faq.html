Frequently Asked Questions

Got Questions?

Find answers to common questions about XOM3, our features, pricing, and more.

General

XOM3 (pronounced "zoom-three") stands for "Xtension of Me" - it's an AI-powered automation platform that helps you build intelligent workflows, automate repetitive tasks, and scale your operations. Think of it as an extension of yourself that works 24/7 to handle the tasks you define.

XOM3 combines the best of visual workflow builders with powerful AI capabilities. Unlike traditional automation tools, XOM3 can:

  • Automatically optimize your workflows for efficiency
  • Use AI agents that understand context and make decisions
  • Adapt to changes in your data and processes
  • Provide intelligent error handling and recovery

No! XOM3 is designed for both technical and non-technical users. Our visual workflow builder lets you create automations by dragging and dropping nodes. However, if you're a developer, you'll love our full API access, SDKs in 8 languages, and the ability to write custom code nodes.

Almost anything! Common use cases include:

  • Data synchronization between apps
  • Lead nurturing and CRM automation
  • Social media scheduling and monitoring
  • Report generation and distribution
  • Customer support ticket routing
  • E-commerce order processing
  • AI-powered content generation
  • DevOps and deployment workflows

Technical

XOM3 AI agents are intelligent nodes that can process unstructured data, make decisions, and generate content. They support multiple AI models including GPT-4, Claude, and custom fine-tuned models. You can configure agents with custom prompts, memory, and tool access to perform complex tasks like analyzing documents, categorizing data, or generating personalized responses.

We offer 200+ native integrations including Slack, Google Workspace, Microsoft 365, Salesforce, HubSpot, Stripe, Shopify, Notion, Airtable, PostgreSQL, MongoDB, and many more. Plus, you can connect to any REST API using our HTTP Request node or create custom integrations with our SDK.

Enterprise customers can deploy XOM3 in their own infrastructure. We support deployment on AWS, GCP, Azure, or on-premises. Self-hosted deployments include all features plus additional security controls and compliance certifications. Contact our sales team for details.

Rate limits vary by plan:

  • Starter: 100 requests/minute
  • Pro: 1,000 requests/minute
  • Enterprise: Custom limits

If you need higher limits, contact us to discuss your requirements.

XOM3 provides multiple error handling options:

  • Automatic retries: Configure retry attempts with exponential backoff
  • Error branches: Create alternative paths when nodes fail
  • Notifications: Get alerts via Slack, email, or webhook
  • Dead letter queues: Capture failed executions for later review
  • AI recovery: Let AI agents attempt to fix common errors

Security & Compliance

Security is our top priority. XOM3 uses:

  • End-to-end encryption (AES-256) for data at rest and in transit
  • Zero-trust architecture with strict access controls
  • Regular third-party security audits
  • Encrypted credential storage with automatic rotation
  • Geographic data residency options

Yes! XOM3 is SOC 2 Type II certified. We also comply with GDPR, CCPA, and HIPAA (for Enterprise customers). Compliance reports are available upon request for customers on Pro and Enterprise plans.

All credentials are encrypted using AES-256 and stored in a dedicated secrets manager. Credentials are never exposed in logs, exports, or API responses. We support OAuth 2.0 for services that offer it, and you can rotate credentials at any time without disrupting active workflows.

Our uptime SLAs vary by plan:

  • Starter: 99% uptime
  • Pro: 99.9% uptime with credits for downtime
  • Enterprise: 99.99% uptime with custom SLA terms

Check our status page for real-time availability.
