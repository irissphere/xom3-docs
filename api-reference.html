API Reference

The XOM3 API is organized around REST. Our API has predictable resource-oriented URLs, accepts JSON-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

Base URL
https://api.xom3.org/v1

Authentication

The XOM3 API uses API keys to authenticate requests. You can view and manage your API keys in the XOM3 Dashboard.

Authentication is performed via HTTP Bearer auth. Include your API key in the Authorization header:

HTTP Header 
Authorization: Bearer xom3_sk_live_xxxxxxxxxxxxx

All API requests must be made over HTTPS. Calls made over plain HTTP will fail. API requests without authentication will also fail.

Rate Limits

The XOM3 API implements rate limiting to ensure fair usage and stability:

Plan Requests/minute Requests/day
Free 60 1,000
Pro 300 50,000
Enterprise Custom Unlimited

Rate limit information is included in response headers:

Response Headers 
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 58
X-RateLimit-Reset: 1640995200

Error Handling

XOM3 uses conventional HTTP response codes to indicate the success or failure of an API request:

Code Description
200 Success - Request completed successfully
201 Created - Resource created successfully
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
403 Forbidden - Insufficient permissions
404 Not Found - Resource doesn't exist
429 Too Many Requests - Rate limit exceeded
500 Server Error - Something went wrong

List Workflows

GET /v1/workflows

Returns a list of all workflows in your account.

Query Parameters

Parameter Type Description
limit integer Maximum number of workflows to return (default: 20, max: 100)
offset integer Number of workflows to skip (for pagination)
status string Filter by status: active, inactive, draft

Example Request

cURL 
curl https://api.xom3.org/v1/workflows \
  -H "Authorization: Bearer xom3_sk_live_xxxxx"

Example Response

JSON 
{
  "data": [
    {
      "id": "wf_abc123",
      "name": "Customer Support Agent",
      "description": "AI-powered customer support",
      "status": "active",
      "created_at": "2024-01-15T10:30:00Z",
      "updated_at": "2024-01-15T10:30:00Z",
      "executions_count": 1523
    }
  ],
  "total": 1,
  "limit": 20,
  "offset": 0
}

Create Workflow

POST /v1/workflows

Creates a new workflow with the specified configuration.

Request Body

JSON 
{
  "name": "Customer Support Agent",
  "description": "AI-powered customer support automation",
  "triggers": [
    {
      "type": "webhook",
      "config": {
        "path": "/support",
        "method": "POST"
      }
    }
  ],
  "actions": [
    {
      "type": "ai_classify",
      "config": {
        "categories": ["billing", "technical", "general"]
      }
    },
    {
      "type": "ai_respond",
      "config": {
        "model": "gpt-4",
        "systemPrompt": "You are a helpful support agent..."
      }
    }
  ]
}

Get Workflow

GET /v1/workflows/{workflow_id}

Retrieves the details of an existing workflow.

Update Workflow

PUT /v1/workflows/{workflow_id}

Updates an existing workflow with new configuration.

Delete Workflow

DELETE /v1/workflows/{workflow_id}

Permanently deletes a workflow. This action cannot be undone.

Trigger Workflow

POST /v1/workflows/{workflow_id}/trigger

Manually triggers a workflow execution with the provided input data.

Example Request

cURL 
curl -X POST https://api.xom3.org/v1/workflows/wf_abc123/trigger \
  -H "Authorization: Bearer xom3_sk_live_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "I need help with my billing",
    "customer": {
      "email": "john@example.com",
      "name": "John Doe"
    }
  }'

List Executions

GET /v1/executions

Returns a list of workflow executions.

Get Execution

GET /v1/executions/{execution_id}

Create AI Agent

POST /v1/agents

Creates a new AI agent with the specified configuration.

Example Request

JSON 
{
  "name": "Sales Assistant",
  "model": "gpt-4",
  "system_prompt": "You are a helpful sales assistant...",
  "tools": ["search_products", "check_inventory", "create_quote"],
  "temperature": 0.7
}

Chat with Agent

POST /v1/agents/{agent_id}/chat

Send a message to an AI agent and receive a response.

Example Request

cURL 
curl -X POST https://api.xom3.org/v1/agents/agent_xyz/chat \
  -H "Authorization: Bearer xom3_sk_live_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What products do you have under $100?",
    "session_id": "sess_abc123"
  }'

Example Response

JSON 
{
  "id": "msg_def456",
  "agent_id": "agent_xyz",
  "session_id": "sess_abc123",
  "message": "I found 15 products under $100! Here are the top picks...",
  "tools_used": ["search_products"],
  "tokens": {
    "input": 45,
    "output": 128
  },
  "created_at": "2024-01-15T10:35:00Z"
}
